# 🌍 გლობალური განგაშის მონიტორი / Global Alert Monitor

ვებ-საიტი, რომელიც რეალურ დროში აკვირდება მსოფლიო მასშტაბის განგაშებს და აჩვენებს მათ ინტერაქტიულ რუკაზე.

## მონაცემთა წყაროები (ცოცხალი, უფასო API-ები)

| წყარო | რას აწვდის |
|---|---|
| [USGS](https://earthquake.usgs.gov/) | მიწისძვრები მთელ მსოფლიოში (M2.5+, ბოლო 24 სთ) |
| [NASA EONET](https://eonet.gsfc.nasa.gov/) | ხანძრები, შტორმები, ვულკანები, წყალდიდობები |
| [GDACS](https://www.gdacs.org/) | გაერო/ევროკომისიის კატასტროფების განგაშები (Orange/Red) |
| [NWS](https://www.weather.gov/) | აშშ-ის ამინდის მძიმე განგაშები (Severe/Extreme) |

## ფუნქციები

- 🗺️ ინტერაქტიული მუქი რუკა (Leaflet + CARTO), მარკერები კატეგორიის ფერით და სიმძიმის ზომით
- 🔄 ავტომატური განახლება ყოველ 60 წამში
- 🔔 ბრაუზერის შეტყობინებები ახალ მნიშვნელოვან განგაშებზე + ეკრანზე toast-ები
- 🏷️ კატეგორიების ფილტრები (მიწისძვრა, ხანძარი, შტორმი, ვულკანი, წყალდიდობა, ამინდი)
- 📊 ცოცხალი სტატისტიკა (კრიტიკული / საშუალო / სულ)
- 📱 ადაპტური დიზაინი მობილურისთვის

## გაშვება

ეს არის ერთი სტატიკური `index.html` — backend არ სჭირდება:

```bash
# უბრალოდ გახსენით ბრაუზერში, ან:
python3 -m http.server 8000
# → http://localhost:8000/GlobalAlerts/
```

GitHub Pages-ზეც პირდაპირ მუშაობს.

> შენიშვნა: ბრაუზერის შეტყობინებები (Notification API) მოითხოვს HTTPS-ს ან localhost-ს.
